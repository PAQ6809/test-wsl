# Restoration Studio Cloud

Evidence-first media restoration for photos and long videos that the user owns or is authorized to process.

- Live service: https://restoration-studio-cloud.vercel.app
- Durable personal Worker: `worker/`
- Processing checkpoints: every five minutes
- Private cloud control plane: Supabase Auth, RLS, private chunk storage and Edge Functions

## Worker setup

### Windows

```powershell
Invoke-WebRequest https://raw.githubusercontent.com/PAQ6809/test-wsl/main/worker/install-worker.ps1 -OutFile install-worker.ps1
.\install-worker.ps1
```

Then copy the one-time pairing token generated by the website into `worker/.env` and run:

```powershell
.\worker\.venv\Scripts\python.exe .\worker\run.py
```

### Docker

```bash
cd worker
cp .env.example .env
# Fill WORKER_TOKEN in .env
docker compose up -d --build
```

## Safety boundary

This project improves compression artifacts, noise, interlacing, sharpness and output resolution. It does not claim to recover pixels that were completely deleted, blacked out, or intentionally hidden by privacy mosaic. Generated details are hypotheses, not evidence.
