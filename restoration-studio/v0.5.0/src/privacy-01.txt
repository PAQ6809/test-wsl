<h2>法律依據與使用目的</h2><p>我們僅為提供帳號、可靠上傳、工作協調、安全稽核、錯誤復原和成品存取而處理資料。不出售個人資料，也不以使用者媒體訓練模型。</p>
<h2>資料隔離與安全</h2><p>公開資料表與私人 Storage Bucket 使用 Row Level Security；使用者只能存取其帳號所擁有的列與以使用者 ID 為首層路徑的檔案。Personal Worker Token 以 SHA-256 雜湊保存，原始 Token 只在建立時顯示一次。</p>
<h2>保留期限</h2><p>工作預設標示七天保留期限；使用者可提前刪除工作、雲端成品與分段。Personal Worker 上的本機內容由使用者自行管理，雲端刪除不會碰觸本機硬碟。</p>
<h2>刪除帳號</h2><p>帳號頁可立即清除私人雲端物件、工作、分段紀錄、Worker Token與產品資料，並建立 Auth 主帳號刪除申請。由於身分供應商管理層需要最終刪除權限，主登入紀錄可能在申請後短暫保留，但不再保有產品媒體與處理紀錄。</p>
<h2>第三方服務</h2><p>目前使用 Supabase 提供帳號、資料庫與私人檔案儲存，Vercel 提供網站託管。各服務亦依其基礎設施政策處理必要的連線與安全日誌。</p>
<h2>聯絡與權利</h2><p>使用者可在產品內存取、更正、匯出或刪除自己的工作資料。安全漏洞請依<a href="/security.html">安全說明</a>中的方式回報。</p></article></main></body></html>
